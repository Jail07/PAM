# Personal Account Management System

## Description
This project implements a personal bank account management system using Python and
Object-Oriented Programming (OOP) principles.

## Classes
### Amount
Represents a transaction with amount, timestamp, and transaction type.

### PersonalAccount
Represents a bank account that supports deposits, withdrawals, balance tracking,
and transaction history.

## How to Run
1. Clone the repository
2. Run the main program:
   python main.py

## How to Run Tests
python test.py
